# .NET CLI
```
dotnet add package Natesworks.Dotmenu
```
# Package Manager Console in Visual Studio
```
NuGet\Install-Package Natesworks.Dotmenu -Version 1.x.x
```
# PackageRefrence
```
<PackageReference Include="Natesworks.Dotmenu" Version="1.x.x" />
```
# Paket
```
paket add Natesworks.Dotmenu --version 1.x.x
```
# Script & Interactive
```
#r "nuget: Natesworks.Dotmenu, 1.x.x"
```
# Cake
```
#addin nuget:?package=Natesworks.Dotmenu&version=1.x.x
```
```
#tool nuget:?package=Natesworks.Dotmenu&version=1.x.x
```
