title: Installation
---

# .NET CLI
```
dotnet add package Dotmenu
```
# Package Manager Console in Visual Studio
```
NuGet\Install-Package Dotmenu -Version 1.0.0
```
# PackageRefrence
```
<PackageReference Include="Dotmenu" Version="1.0.0" />
```
# Paket
```
paket add Dotmenu --version 1.0.0
```
# Script & Interactive
```
#r "nuget: Dotmenu, 1.0.0"
```
# Cake
```
#addin nuget:?package=Dotmenu&version=1.0.0
```
```
#tool nuget:?package=Dotmenu&version=1.0.0
```
