## Builders

Currently, there are a few builders available to help you create menus:

`Menu.CreateDefaultBuilder()`
`Menu.CreateCenteredBuilder()`

## Creating menu

### Storing menu in variable

```cs
var menu = Menu.CreateDefaultBuilder();
menu.SetTitle("Sample");
menu.SetPrefix("");
menu.SetSelector(">");
menu.AddOption(text: "Exit", action: Exit);

menu.Build().Run();
```

### Chaining

```cs
Menu.CreateDefaultBuilder()
    .SetTitle("Example")
    .SetPrefix("")
    .SetSelector(">")
    .AddOption(text: "Exit", action: Exit)
    .Build()
    .Run();
```

## Setting title

The title (formerly prompt) is the text shown before the menu options.

```cs
.SetTitle("Sample")
```

## Setting prefix

The prefix is the text shown before an option name when it is not selected.

```cs
.SetPrefix("")
```

## Setting selector

The prefix is the text shown before an option name when it is selected.

```cs
.SetSelector(">")
```

## Add option

```cs
.AddOption(text: "Exit", action: Exit)
```

## Actions

An action is the method that will run when option is selected.

```cs
static void Exit() =>
    Environment.Exit(exitCode: 0);
```

## Seperator

The seperator can be used to seperate menu options.

```cs
.AddSeparator('-', length: 10)
```

## Building and running

```cs
Build().Run();
```
