---
title: Editing options
---

## Set Text

```cs
menu.option[index].SetText(text);
```

## Set Action

```cs
menu.option[index].SetAction(action);
```

## Set Hidden

```cs
menu.option[index].SetHidden(hidden);
```

## Get Text

```cs
string text = menu.option[index].GetText();
```

## Get Action

```cs
var action = menu.option[index].GetAction();
```

## Get Hidden

```cs
bool hidden = menu.option[index].GetHidden();
```
