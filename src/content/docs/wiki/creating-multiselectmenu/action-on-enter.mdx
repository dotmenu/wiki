---
title: Action on Enter
---

To set the action to run when enter is pressed, use the `ActionOnEnter` method.

```csharp
menu.ActionOnEnter(() =>
{
    Console.WriteLine("Selected(tab) options:");
    foreach (var index in menu.GetCheckedOptions())
    {
        Console.WriteLine($"Option {index + 1}");
    }
});
```
